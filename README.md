# kod
1. preprocess.py - bierze dane oryginalne(raw_data) i wypluwa dane obrobione(preprocessed_data)
2. models.py - trenuje modele i robi jakąś crossvalidację czy cośtam, wypluwa plik z podsumowaniem
3. predict.py - trenuje wybrany model na całym zbiorze train i robi predykcje gotowe do wysłania na zbiorze test
4. EDA.ipynb - zawiera exploratory data analysis, wykresy i całą naszą wiedzę o danych
